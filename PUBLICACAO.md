# Checklist de Publicacao

## 1. Criar repositorio

Nome sugerido:

```text
cocriacao-assincrona-extensao
```

## 2. Enviar arquivos

Arquivos principais:

- `index.html`
- `README.md`
- `notion-template.md`
- `PUBLICACAO.md`

## 3. Ativar GitHub Pages

No GitHub:

1. Abrir `Settings`.
2. Entrar em `Pages`.
3. Em `Build and deployment`, selecionar `Deploy from a branch`.
4. Selecionar branch `main`.
5. Selecionar pasta `/ (root)`.
6. Salvar.

## 4. Conectar com Notion

1. Criar o mural no Notion usando `notion-template.md`.
2. Ajustar permissoes de acesso para os membros da equipe.
3. Copiar o link compartilhado.
4. Abrir o HTML publicado.
5. Entrar na aba `Notion`.
6. Colar e salvar o link do mural.

## 5. Decisao antes da versao publica

Definir se o link do Notion sera:

- privado, apenas para equipe;
- publico para visualizacao;
- publico apenas como pagina curada, sem permitir edicao externa.
