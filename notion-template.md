# Mural de Ideias | Projeto de Extensao

Use este modelo para criar uma pagina/database no Notion.

## Nome da pagina

Etapa 1 - Co-criacao Assincrona e Descentralizada

## Visao principal

Tipo: Board / Kanban

Agrupar por: `Categoria`

## Propriedades sugeridas

| Propriedade | Tipo | Uso |
| --- | --- | --- |
| Titulo | Title | Nome curto da ideia, referencia ou acao |
| Categoria | Select | Coluna do mural |
| Status | Select | Aberta, Em maturacao, Adotada, Em execucao, Concluida, Pausada |
| Autor | Person/Text | Quem registrou a ideia |
| Responsavel | Person/Text | Quem adotou ou esta conduzindo |
| Frente | Multi-select | Educacao, Turismo, Campo, Dados, Hub, Escolas, Gestao publica |
| Prioridade | Select | Baixa, Media, Alta |
| Apoios | Number | Quantidade de membros interessados na ideia |
| Comentarios-chave | Text | Perguntas, complementos e sinteses importantes |
| Ciclo | Select/Text | Ex: Ciclo 01, Ciclo 02 |
| Proximo passo | Text | Acao pequena e concreta |
| Referencias | URL/Text | Links, imagens, artigos ou materiais |
| Data de criacao | Created time | Registro automatico |
| Ultima atualizacao | Last edited time | Registro automatico |

## Categorias do mural

- O que ja temos
- Ideias livres
- Referencias e inspiracoes
- Ideias em maturacao
- Ideias adotadas
- Proximas acoes
- Sintese dos ciclos

## Cartoes iniciais sugeridos

### Hub digital e apps de campo

Categoria: O que ja temos

Descricao: Centralizar os links do Hub, repositorios, apps de levantamento, termo de conduta, mapas e materiais ja desenvolvidos.

### Briefing ecologico para operadoras

Categoria: Ideias livres

Descricao: Criar um roteiro curto para guias, tripulacoes e dive masters apresentarem macroalgas, costoes, seguranca e literacia oceanica antes das atividades.

### Guia visual impermeavel

Categoria: Ideias em maturacao

Descricao: Testar formato de prancheta para uso embarcado e subaquatico, com especies-chave e observacoes simples.

### Sintese do ciclo 01

Categoria: Sintese dos ciclos

Descricao: Consolidar 3 a 5 encaminhamentos apos as primeiras contribuicoes da equipe.

## Regra simples de comunicacao

O chat avisa que algo mudou. O Notion guarda a informacao.

## Como cada membro participa

1. Registrar uma ideia, referencia, demanda ou duvida.
2. Comentar ideias de outras pessoas com perguntas e complementos.
3. Sinalizar apoio quando uma proposta parecer importante.
4. Adotar voluntariamente uma ideia quando houver condicao de desenvolver.
5. Atualizar o proximo passo de forma pequena e objetiva.

## Rotina sugerida

- Semana 1: adicionar ideias, referencias e perguntas.
- Semana 2: comentar, agrupar, priorizar e adotar ideias.
- Fechamento: publicar uma sintese curta com prioridades, responsaveis voluntarios e proximos passos.
